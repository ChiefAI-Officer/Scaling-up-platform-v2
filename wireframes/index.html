Scaling Up Assessment Tool

Wireframes for the admin assessment-creation wizard and campaign detail. Built May 6, 2026 from the locked v7.1 implementation plan. These mockups are for stakeholder review before any production code is written.

Plan version: v7.1 Scope: Wizard + Campaign Detail (10 screens) Stakeholder: Jeff Verdun
How to read these wireframes. Every screen has a yellow "WIREFRAME" banner at the top — clicking interactive elements will not do anything. Mock data is realistic but illustrative only. The visual style mirrors the existing Scaling Up Platform v2 (same color tokens, typography, and component patterns). Each screen cites the specific section of the v7.1 plan it implements (e.g., campaign-service.ts → createCampaign).
Assessment Creation Wizard
1

Step 1 — Template & Organization

Pick the assessment template. Four INVITED options (Rockefeller, Vision Alignment, QSP v2, Scaling Up) and two PUBLIC options (SunHub Quick Quiz, Website Assessment). Organization is locked from breadcrumb context.

v7.1 ref: campaign-service.ts → createCampaign
2

Step 2 — Participants INVITED only

Two-pane layout. Left: collapsible team-tree picker (Acme > Engineering > Platform). Right: flat list of selected members with team breadcrumbs. Plus an ad-hoc add form. Skipped entirely for PUBLIC templates.

v7.1 ref: addParticipants, OrgTeam recursive hierarchy, teamPathAtAdd
3

Step 3 — Schedule & Access Mode

Campaign name + description, openAt / closeAt or open-ended (default for PUBLIC), notification toggle. Access mode selector (INVITED / PUBLIC) drives whether Step 3a appears next.

v7.1 ref: AssessmentCampaign fields, expiresAt rule, evergreen PUBLIC
3a

Step 3a — PUBLIC Config PUBLIC only

Lead-capture form fields (email locked, name/company/job title/referring coach toggleable), results routing (taker / referring coach if registered / always-CC list), CTA buttons with optional tier gating.

v7.1 ref: publicConfig.leadCapture / resultsRouting / ctaButtons, referringCoachEmail allowlist
4

Step 4 — Review & Create

Collapsible summary of every prior step. "What happens next" copy adapts to access mode. PUBLIC review includes copy-paste iframe embed code. Save as Draft or Create & Activate.

v7.1 ref: createCampaign + optional inviteRespondents

Respondents Tab + Inline Expand

Tab strip (Overview / Respondents / Aggregate / Trends). Per-respondent table with chevron expand. One row shown expanded mid-table with summary; "View full report →" links to screen 8.

Individual Results View

Team Aggregate Panel

Team-scope dropdown, top stat row (Submissions, Avg Score, Tier Distribution, Highest Tier). Per-question stacked bars showing 0/1/2/3 spread across submissions. Per-section average bars (S1–S10).
