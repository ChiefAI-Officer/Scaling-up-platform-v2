v7.5 API surface

  • POST /api/coaches/[coachId]/template-grants — body { templateId } — creates a TemplateAccessGrant row. Returns 409 if grant already exists. Auth: ADMIN/STAFF.
  • DELETE /api/coaches/[coachId]/template-grants/[grantId] — revokes. Auth: ADMIN/STAFF. Idempotent (404 if not found).
  • GET /api/coaches/[coachId]/template-grants — list grants for a coach (used by Wireframe 12 Template Access Grants card). Returns INVITED-mode grants only.

v1 default: 4 INVITED templates auto-grant on coach creation. PUBLIC templates (Website Assessment, future SunHub) are platform-managed and excluded from coach grants.

Wireframe end note. Bulk grant + bulk revoke + per-template audit log views ship in v1.5+. v1 surface: this matrix + the per-coach “Template Access Grants” card on Wireframe 12.
